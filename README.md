# Portfolio — Benjamin Hansen

Portfolio personnel **statique**, écrit à la main en HTML, CSS et un peu de JavaScript (sans framework).

## Structure

- `index.html` — structure et contenu
- `style.css` — mise en forme
- `script.js` — interactivité (année du pied de page)
- `public/` — CV et favicon

## Déploiement

Déployé sur **Vercel** (déploiement automatique à chaque push sur `master`), et auto-hébergé
sur mon homelab derrière un reverse proxy Nginx.

🔗 https://benjamin-hansen.vercel.app
